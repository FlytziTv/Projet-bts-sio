# Projet Sudoku

Ce projet est une application web de Sudoku réalisée en **HTML**, **CSS** et **JavaScript** dans le cadre de la Piscine Discovery BTS 1 à Ynov Campus Paris, encadrée par Armand.L (toredux).

## Fonctionnalités

- Génération d'une grille de Sudoku
- Interface utilisateur interactive
- Validation des solutions
- Design responsive

## Installation

1. Clonez le dépôt :
   ```powershell
   git clone https://github.com/FlytziTv/Projet-bts-sio.git
   ```
2. Ouvrez le dossier `sudoku`.
3. Lancez `index.html` dans votre navigateur.

## Utilisation

Remplissez la grille de Sudoku directement dans votre navigateur. Utilisez les boutons pour vérifier votre solution ou réinitialiser la grille.

## Technologies utilisées

- HTML5
- CSS3
